# 🛒 E-Commerce Sales Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing e-commerce sales data to uncover meaningful business insights. The goal is to transform raw data into an interactive dashboard that helps in better decision-making.

---

## 🎯 Objectives
- Analyze sales performance across different categories and regions  
- Identify top-selling products and high-profit segments  
- Track monthly sales and profit trends  
- Build an interactive dashboard for easy data visualization  

---

## 📊 Dataset
- Contains 500+ rows of e-commerce sales data  
- Includes fields like Order Date, Category, Region, Sales, Profit, Quantity  

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel  
  - Pivot Tables  
  - Pivot Charts  
  - Data Cleaning  
  - Formulas & Functions  
  - Slicers (for interactivity)  

---

## ⚙️ Process
1. Data Cleaning  
   - Removed duplicates and handled missing values  
   - Formatted date columns and extracted Month  

2. Data Transformation  
   - Created calculated columns (Month, Profit Analysis)  

3. Data Analysis  
   - Built Pivot Tables for category-wise, region-wise, and monthly analysis  

4. Dashboard Creation  
   - Designed interactive dashboard using charts and slicers  

---

## 📈 Key Insights
- Identified top-performing product categories  
- Found highest sales-generating regions  
- Discovered peak sales months  
- Highlighted profit vs sales trends
